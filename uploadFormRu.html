Шаг 1.

Здесь происходит загрузка файлов, составляющих цифровую коллекцию.
Можно выбрать несколько индивидуальных файлов или папку.

Ограничения

- Максимум 500 файлов.

- Принимаемые типы файлов: PDF, PNG, JPG, JPEG.

- Суммарный объём коллекции не должен превышать 5 GB.

- Размер одного файла не должен превышать 10 MB.

- У каждого файла должно быть уникальное имя.

Шаг 2.

Здесь происходит загрузка CSV-файла с метаданными, описывающими все элементы коллекции.
Одна строка таблицы = один загруженный файл.

Ограничения

- Только один файл.

- Принимаемые типы файлов: CSV (comma-separated values).

- Размер одного файла не должен превышать 10 MB.

- У каждой записи в таблице должен быть уникальный идентификатор (id), полностью совпадающий с именем соответствующего файла.

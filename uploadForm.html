Step 1.

Here you upload a selection of files that represent your collection.
You can choose multiple individual files or a folder.

Limitations

- 500 files maximum.

- Accepted file types: PDF, PNG, JPG, JPEG.

- Total collection size should not exceed 5 GB.

- One file should not exceed 10 MB.

- Each file should have a unique name.

Step 2.

Here you upload a CSV-file that describes your collection.
One row in the table = one previously uploaded file.

Limitations

- One file only.

- Accepted file type: CSV (comma-separated values).

- One file should not exceed 10 MB.

- Each entry should have a unique id that is exactly the same as the corresponding file name.
